# Local Events – Django Web Application

Web application for managing and discovering local events.

- Django (Backend)
- Bootstrap (Frontend)
- Figma (UI Design)

## Postavuvanje na 5 zadaci i nivna implementacija
- User authentication
- Create, edit, delete events
- Search and filters
- Pagination
- Permission control (owner-only actions)
